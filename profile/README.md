# Oregonstate AI

To join this Github Organization, please [login here with your ONID](https://github-identity-bridge-app.azurewebsites.net/api/login)

Oregonstate-ai Github is managed by the ARCS team at Oregon State University and contains research computing code, configurations, and documentation. Please do not store any data in a repository inside this organization. Any attempts to store data (e.g., in CSV files) will be monitored, and data will be removed. Please do not store any master data such as IP addresses, hostnames, or any other sensitive network information in any repository in this organization. Please store this kind of master data in an approved environment such as Azure Key Vault. And this goes without saying: Never, ever store any restricted or sensitive information in these repositories
